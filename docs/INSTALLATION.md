# Installation Guide

## Requirements
- Node.js 20+
- npm 10+
- Chrome/Chromium

## Local Setup
```bash
cd /home/runner/work/cses-codeforces-code-pusher/cses-codeforces-code-pusher/extension
npm install
npm run build
```

## Load Extension
1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select `/home/runner/work/cses-codeforces-code-pusher/cses-codeforces-code-pusher/extension/dist`

## Configure
1. Add GitHub token in extension settings
2. Select repository owner/name/branch
3. Enable Auto Sync
